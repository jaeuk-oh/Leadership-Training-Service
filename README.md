# GROW 코칭 시뮬레이터

> AI 페르소나와의 실전 코칭 연습으로 리더십 역량을 키우는 트레이닝 서비스

---

## 프로젝트 개요

리더십 코칭 교육의 가장 큰 한계는 **"연습할 상대"가 없다는 것**입니다. 실제 코칭 상황을 경험하려면 현장에 나가야 하고, 잘못된 접근 방식을 반복해도 즉각적인 피드백을 받기 어렵습니다.

GROW 코칭 시뮬레이터는 이 문제를 해결하기 위해 만든 AI 기반 코칭 실습 플랫폼입니다. 국제 코칭 연맹(ICF)이 권장하는 **GROW 모델**(Goal → Reality → Options → Will)을 기반으로, 실제 직장 상황을 반영한 AI 페르소나와 1:1 코칭 대화를 나누고, 세션 종료 후 GPT-4o가 코칭 역량을 정량 평가합니다.

개인 사이드 프로젝트로 기획부터 배포까지 **풀스택 단독 개발**하였으며, AI 파이프라인 설계와 RAG 구현에 특히 집중했습니다.

## 기술 스택

| 구분 | 기술 |
|------|------|
| **Frontend** | Next.js 15 (App Router), TypeScript, Tailwind CSS, shadcn/ui |
| **Backend** | FastAPI (Python), Uvicorn |
| **AI / LLM** | OpenAI GPT-4o (대화·평가), GPT-4o-mini (신뢰도 평가), Whisper-1 (STT), TTS-1 (음성 합성) |
| **Database** | Supabase (PostgreSQL + pgvector) |
| **RAG** | text-embedding-3-small + pgvector 벡터 유사도 검색 |
| **Deployment** | Vercel (Frontend), Railway (Backend) |
| **CI/CD** | GitHub Actions |

**핵심 설계 포인트:**
- 채팅 응답은 **SSE(Server-Sent Events)** 스트리밍으로 제공해 체감 응답 속도 개선
- 신뢰도 평가(gpt-4o-mini)와 AI 응답 생성(gpt-4o)을 **asyncio 병렬 처리**로 레이턴시 단축
- 페르소나 성격/상황 정보는 **카테고리별 문서로 분리 저장 후 벡터 임베딩**, RAG로 질의 관련 정보만 컨텍스트에 주입

## 주요 기능

### 1. GROW 코칭 세션
AI 페르소나와 GROW 4단계 구조의 코칭 대화를 실시합니다. 코치(사용자)의 질문 품질에 따라 피코치자의 **신뢰도(0~10점)**가 실시간으로 변화합니다.

- 세션 시작 시 AI 페르소나가 먼저 자기소개와 현재 고민을 제시해 자연스러운 대화 진입
- 단계 전환 버튼(G → R → O → W)으로 세션 흐름 직접 제어
- 신뢰도가 낮으면 AI가 방어적·폐쇄적으로, 높으면 개방적으로 반응

**신뢰도 평가 기준 (GPT-4o-mini, -2.0 ~ +2.0/턴)**

| 기준 | 설명 |
|------|------|
| 공감적 경청 | 상대방의 감정·상황 인정 여부 |
| 열린 질문 | Yes/No가 아닌 탐색적 질문 사용 |
| GROW 단계 적합성 | 현재 단계에 맞는 질문·접근 |
| 판단/조언 자제 | 직접적 해답 제시 자제 |

### 2. GROW 가이드 사이드 패널
채팅 화면 우측에 **상시 노출되는 가이드 패널**을 제공합니다.

- 현재 GROW 단계에 맞는 예시 질문을 패널에서 바로 확인
- 질문 클릭 시 입력창에 자동 삽입 — 가이드를 보며 즉시 전송 가능
- 토글 버튼으로 패널 접기/펼치기

### 3. 음성 입출력 (STT / TTS)
- **STT**: 마이크 버튼으로 음성 녹음 → Whisper-1 전사 → 자동 전송
- **TTS**: 응답 텍스트를 OpenAI TTS-1으로 합성해 페르소나 목소리로 재생
- 페르소나마다 다른 voice(alloy, nova, echo 등) 설정 가능

### 4. RAG 기반 페르소나 시스템
각 페르소나는 프로필·말투·업무상황·심리상태·대화반응·배경스토리의 **6개 카테고리 문서**로 구성됩니다. 코칭 메시지가 들어오면 pgvector로 관련 문서를 검색해 시스템 프롬프트에 동적으로 주입, 일관성 있는 페르소나를 유지합니다.

- 페르소나별 고유 색상 아바타 및 이름 표시로 몰입감 강화
- 응답 대기 시 페르소나 아바타와 함께 타이핑 인디케이터 표시

### 5. AI 코칭 역량 평가
세션 종료 시 GPT-4o가 전체 대화를 분석해 4가지 지표를 0~10점으로 채점합니다.

| 지표 | 설명 |
|------|------|
| 목표 명확화 | SMART 목표로 이끄는 질문 능력 |
| 경청/공감 | 감정 반영·요약·공감 표현 |
| 질문 수준 | 열린 질문·탐색적 질문·강력한 질문 |
| 실행 의지 | 구체적 행동 계획·책임감 부여 |

레이더 차트, 신뢰도 비교 바 차트, GPT 개선 피드백 텍스트로 시각화됩니다.

### 6. 랭킹 & 대시보드
- 신뢰도 점수 기반 **TOP 10 리더보드**
- 세션 이력, 평균 신뢰도, 최고 점수를 한눈에 확인하는 개인 대시보드

### 7. 커스텀 페르소나 생성
관리자 또는 사용자가 성별·나이·직군·MBTI·스트레스 수준 등 세부 속성을 입력해 나만의 페르소나를 생성하고 즉시 임베딩할 수 있습니다.

## 결과 및 성과

- GPT-4o 스트리밍 응답으로 **체감 응답 시간 대폭 단축** (전체 응답 완료 대비 첫 토큰까지 ~500ms 내)
- 신뢰도 평가와 메인 응답 생성을 병렬 처리해 **세션 종료 시 추가 대기 시간 제거**
- RAG 임계값(0.65) 튜닝으로 **관련 없는 컨텍스트 주입 최소화**, 페르소나 일관성 향상
- 프리셋 페르소나 3종 제공(쉬움/보통/어려움), 난이도별 코칭 실습 경로 구성
